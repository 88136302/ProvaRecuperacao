Rogue Wave Software compra empresa líder em PHP, Zend Technologies

A Rogue Wave Software anunciou hoje que adquiriu a empresa Zend Technologies, líder em soluções de desenvolvimento PHP web e mobile. Com 50% da web rodando PHP (inclusive Magento, Drupal e WordPress), os produtos da Zend levam o PHP para a Rogue Wave, desde a criação de códigos até o deploy.

“O anúncio de hoje expande a Rogue Wave para o mundo do desenvolvimento de aplicações PHP para web e mobile, sublinhando o nosso objetivo de criar desenvolvedores heróis, acelerando a capacidade de criar ótimos código”, disse Brian Pierce, CEO da Rogue Wave. “Com a Zend, agora temos produtos que aceleram o desenvolvimento em C, C ++, C #, Java e PHP, refletindo como o software é criado hoje em dia em várias linguagens, plataformas e equipes”.

Andi Gutmans, co-fundador e CEO da Zend disse: “Nossa paixão sempre foi os usuários de PHP. Quando eu fundei a Zend, nós decidimos facilitar para os desenvolvedores o uso do PHP para atender às demandas de desenvolvimento de negócios. Estamos muito felizes com esta parceria com a Rogue Wave, com o nosso compromisso com os clientes e em como podemos afetar seu ciclo de vida de desenvolvimento de software. A força desta combinação beneficia os nossos clientes e expande nossa capacidade de oferecer produtos que são importantes para nossos usuários”.

Desenvolvimento Web

Desenvolvimento web é o termo utilizado para descrever o desenvolvimento de sites, na Internet ou numa intranet. O desenvolvimento refere-se a um processo de construção e testes do software especifico para a web, com a finalidade de se obter um conjunto de programas, que satisfazem as funções pretendidas, quer em termos de usabilidade dos usuários ou compatibilidade com outros programas existentes.
